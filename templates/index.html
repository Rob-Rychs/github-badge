What's new?

With the brand new 2nd version of GitHub Badge, now you have a totally renewed UI. We also have the following list of new features:

  • A sparkline showing the latest commit activity on your repos (#8)
  • A tagline showing the project you have worked on most in the last 10 days (#20)
  • Separated repo count for forked repos
  • Shortened follower counts for compactness (#27)
  • Ability to turn off Google Analytics code on your badge (#15)
  • Ability to promote GitHub badge, on your badge (#17)
  • A brand new multi-threaded Python 2.7 backend on Google App Engine (#21, #29)
  • At most one day old statistics on your badge (#24)

What are you waiting for? Give it a spin!

About

GitHub badge is a simple embeddable badge showing your GitHub stats like the number of public repositories, number of followers, favorite languages etc. It runs on Google App Engine with a day-long memcached backup.
